Exploratory Data Analysis of Airbnb Listings in London, UK
Author: Anurag Ganesh Fengade
This project involves performing an Exploratory Data Analysis (EDA) on Airbnb listings in London, UK. The primary goal is to derive insights into pricing, availability, neighborhood trends, and other key metrics that can inform both hosts and guests, as well as data-driven business decisions.

📁 File Structure
eda-airbnb-london-uk.ipynb - The main Jupyter notebook containing the analysis.
data/ - Folder where the Airbnb dataset should be placed (not included in this repo).

📊 Features of the Analysis
Data Cleaning and Preprocessing
Handling Missing Values and Outliers
Univariate and Bivariate Analysis
Neighborhood Distribution and Price Comparison
Availability Trends and Room Type Analysis
Visualization using Matplotlib and Seaborn

🛠️ Technologies Used
Python 3
Pandas
NumPy
Matplotlib
Seaborn

Jupyter Notebook
📌 How to Run
Clone the repository.
Download the dataset from Airbnb or Kaggle and place it in the data/ folder.
Open the notebook using Jupyter and run all the cells sequentially.

📈 Sample Insights
Central London has the highest number of listings and higher average prices.
Entire home/apartment is the most common room type.
Availability varies significantly by neighborhood and listing type.
